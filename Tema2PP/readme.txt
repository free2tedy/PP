Regular Expressions

Implementati functia "mathches", in fisierul "Regexp.hs".

Rulare tester:

> make run

Stergerea fisierelor obiect:

> make clean

Pentru utilizarea fisierului "Makefile", pe Windows, utilizati un adaptor,
cum este Cygwin:

http://www.cygwin.com/

In procesul de instalare, bifati pachetul "make", din grupul "Devel"
(de la "development").
